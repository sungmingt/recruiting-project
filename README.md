# recruiting-project
기업의 채용을 위한 웹 서비스 API

<br>

원티드의 프리온보딩 코스 지원에 필요한 과제를 학습 겸 수행해본 내용입니다.
명시된 요구사항들을 토대로 API를 구현하고, 이후 여건이 된다면 웹 서비스 API로써의 완성도를 높여볼 생각입니다.
